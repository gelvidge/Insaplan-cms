## Design Principles & Brand Guidelines

### Visual Design Strategy

**Color Palette** (From App Mantine Theme)

**Primary Colors**
- **Primary (Deep Blue)**: `#425aa1` → `#060a14` — Core brand color for headers, CTAs, and primary actions
  - Light mode: `#425aa1` (Deep Blue 6)
  - Dark mode: `#060a14` (Deep Blue 9)
- **Secondary (Purple)**: `#9453b5` → `#64317f` — Complementary accent for emphasis
  - Light mode: `#9453b5` (Purple 6)
  - Dark mode: `#64317f` (Purple 9)
- **Default Gradient**: Deep Blue 9 (45°) → Purple 6 — Used for backgrounds, headers, hero sections

**Accent Colors**
- **Red**: `#c75050` — Errors, warnings, negative actions
- **Green**: `#6ab281` — Success, positive actions, confirmations
- **Blue**: `#5f7cb7` — Information, secondary actions, links

**Neutral Palette**
- **Light Gray**: `#f8f9fa` (Gray 0) — Page backgrounds
- **Medium Gray**: `#dee2e6` (Gray 3) — Borders, dividers
- **Dark Gray**: `#495057` (Gray 7) — Body text
- **Black**: `#000000` — Headlines, dark text
- **White**: `#FFFFFF` — Card backgrounds, text on dark backgrounds

**Data Visualization Palette**
- Use all available colors: Red, Green, Blue, Deep Blue, Purple, Pale Purple
- Ensures consistency with app dashboards and charts

**Typography**
- **Font Family**: Open Sans (headlines and body)
- **Monospace**: Space Mono (code, data elements)
- **Headlines (H1-H6)**:
  - H1: 2.125rem, weight 700, line-height 1.3
  - H2: 1.625rem, weight 700, line-height 1.35
  - H3: 1.375rem, weight 700, line-height 1.4
  - H4: 1.125rem, weight 700, line-height 1.45
  - H5: 1rem, weight 700, line-height 1.5
  - H6: 0.875rem, weight 700, line-height 1.5
- **Body**: Regular weight for readability
- **Font Weights**: 500 (semi-bold), 700 (bold), 900 (extra-bold)

**Design Elements**
- Gradient overlays using Deep Blue → Purple gradient for depth
- Animated dashboard previews and feature demonstrations
- Interactive component showcases (tabbed sections with `deepblue.6` highlight color)
- Progress/goal visualization elements (rings, bars, charts in brand colors)
- Micro-animations for engagement without distraction
- Border radius: xs (2px), sm (4px), md (8px), lg (16px), xl (32px)
- Spacing system: xs (10px), sm (12px), md (16px), lg (20px), xl (32px)

### Content & Messaging Strategy

**Hero Section Approach**
- Lead with a powerful problem statement or transformation promise
- Use contrast: "From Manual Planning → Intelligent Execution"
- Include a compelling video demo or animated preview
- Dual CTAs: "Start Free Trial" (low friction) + "Schedule Demo" (enterprise)

**Trust & Credibility Indicators**
- Customer logos and testimonials with measurable results
- Statistics showing impact (time saved, plans created, teams using platform)
- Security/compliance badges (SOC 2, GDPR, ISO 27001, HIPAA)
- User count or adoption metrics

**Feature Demonstration**
- Interactive product previews/screenshots showing real dashboards
- Feature cards in grid layout with short descriptions + "Learn More" links
- Animated feature carousel showing use cases in action
- Before/after comparisons (manual process vs. Insaplan)

**Social Proof & Success**
- Customer success stories with specific metrics (time saved, plan accuracy improved)
- Case studies by industry vertical (Sales, Marketing, Startup, Enterprise)
- Customer testimonials with photos, titles, and company names
- "X companies trust Insaplan" messaging
